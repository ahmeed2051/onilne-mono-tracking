Tech stack

Reliable architecture with realtime at its core.

Pairing a modern PWA frontend with a realtime event-sourced backend keeps the experience fast, synchronized, and auditable.

Frontend

  • React + Vite PWA
  • Tailwind CSS and i18next
  • IndexedDB caching for offline resilience

Backend

  • NestJS API with PostgreSQL + Prisma
  • Redis pub/sub for realtime rooms
  • Event-sourced ledger for auditability

Realtime & Notifications

  • Socket.io channels per game
  • Web Push (VAPID), FCM, and APNs adapters
  • Optimistic UI with server reconciliation

Security & Safety

  • Scoped tokens & role-based permissions
  • Rate limiting and banker lock mode
  • Auto-purge ended games for privacy

Questions

Everything you need to know before the next roll.

Curious how Monopoly Money Tracker fits at your table? Start with these quick answers.

Does every player need an account?

Nope. Hosts can invite guests with a one-time code that expires when the game wraps, while regulars can create reusable profiles.

Can we play without the internet?

The PWA works offline for short stretches and automatically syncs pending moves once you reconnect to Wi‑Fi or cellular data.

How do approvals work?

Enable banker-only or multi-approver modes. Each request shows before/after balances plus notes so everyone trusts the outcome.

Is there a cost to join?

Monopoly Money Tracker is free for casual play. We may add optional pro analytics later, but the core experience stays free.
